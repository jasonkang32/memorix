# Memorix 추가 개발 검토 및 수정 항목

## 신규 기능 검토 (Feature)
1. **기간 지정 갤러리 자동 가져오기**
   - 기간을 입력하면 폰에 있는 갤러리에서 해당 기간 내의 사진과 영상을 자동으로 가져오는 기능. (현재는 수동 선택 위주이므로 대량 임포트 편의성 개선)

## 버그 수정 (Bug Fix)
1. **내용 입력 필드 힌트(Placeholder) 오류 수정**
   - 내용 입력 부분을 터치했을 때 힌트 텍스트가 사라져야 하는데, 사라지지 않는 문제 해결. (TextField의 `hintText` 적용 여부 또는 컨트롤러 상태 점검 필요)
2. **날짜 선택 시 캘린더 위젯 미표시 버그**
   - 날짜를 선택하는 UI에서 달력(캘린더 위젯)이 화면에 나타나지 않는 문제 해결.
3. ~~**사진/영상 통합 등록 시 앱 크래시(종료) 발생**~~ *(v1.0.0+2 완료)*
   - 사진과 영상을 한 메뉴에서 등록하는 과정에서 저장이 되지 않고 앱이 강제 종료되는 문제 수정.
